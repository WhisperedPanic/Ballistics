# 🔫 Interior Ballistics Simulator

A **7-equation lumped parameter model** for interior ballistics simulation, based on Corner/Baer-Frankle formulations. Runs entirely in the browser via GitHub Pages.

## 🌐 Live Demo

**https://whisperedpanic.github.io/Ballistics/**

## 📋 Features

- ✅ 7 Coupled ODEs (projectile, combustion, thermodynamics, Lagrange gradient)
- ✅ RK4 Integration (4th-order Runge-Kutta)
- ✅ 21 Propellants in Library (7 burn rate clusters)
- ✅ Interactive Charts (Chart.js)
- ✅ CSV Export
- ✅ Shareable URLs
- ✅ No Build Step Required

## 🚀 Deployment

1. Create repository on GitHub
2. Push all files
3. Enable GitHub Pages (Settings → Pages → main branch → root)
4. Visit: `https://YOUR_USERNAME.github.io/Ballistics/`
